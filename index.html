Pengenalan Internet

Internet adalah jaringan global yang menghubungkan jutaan jaringan pribadi, publik, akademik, bisnis, dan pemerintah. Internet menggunakan protokol komunikasi standar seperti TCP/IP untuk berbagi informasi dan sumber daya.

Apa Itu HTTP & HTTPS?

HTTP adalah singkatan dari HyperText Transfer Protocol dan merupakan dasar dari pertukaran data di web. Protokol ini memungkinkan pengambilan sumber daya, seperti dokumen HTML.

HTTPS, atau HyperText Transfer Protocol Secure, adalah versi aman dari HTTP, yang berarti bahwa ia mengenkripsi pertukaran data antara klien dan server.

Apa Itu Nama Domain?

Nama domain adalah alamat yang digunakan pengguna untuk mengakses sebuah situs web, seperti "contoh.com". Nama domain adalah bentuk IP address yang dapat dibaca oleh manusia, sehingga memudahkan pengguna dalam menavigasi web.

Bagaimana Cara Kerja Nama Domain?

Saat pengguna memasukkan nama domain ke dalam browser, DNS menerjemahkannya menjadi alamat IP. Alamat IP ini mengarahkan pengguna ke server yang menyimpan konten situs web tersebut.

Apa Itu Hosting?

Hosting adalah layanan yang memungkinkan situs web atau aplikasi web disimpan dan diakses di internet. Hosting melibatkan penyimpanan file situs web pada server yang selalu terhubung ke internet.

Bagaimana Cara Kerja Hosting?

Saat pengguna memasukkan URL situs web, server hosting mengirimkan file situs web ke browser pengguna, yang kemudian menampilkan kontennya. Terdapat beberapa jenis hosting, seperti shared hosting, VPS, dan dedicated hosting.

Apa Itu DNS?

Domain Name System (DNS) adalah seperti buku telepon di internet. DNS menerjemahkan nama domain menjadi alamat IP sehingga browser dapat memuat konten situs web dengan benar.

Bagaimana Cara Kerja DNS?

Saat pengguna memasukkan nama domain, browser menghubungi server DNS untuk menemukan alamat IP yang sesuai. Browser kemudian terhubung ke server dengan alamat IP tersebut untuk memuat halaman web yang diminta.

Apa Itu Browser?

Browser adalah aplikasi perangkat lunak yang memungkinkan pengguna mengakses dan berinteraksi dengan situs web. Beberapa browser populer termasuk Chrome, Firefox, Safari, dan Edge.

Bagaimana Cara Kerja Browser?

Browser mengirimkan permintaan ke server web melalui HTTP/HTTPS. Browser mengambil file HTML, CSS, dan JavaScript dari situs web, menafsirkannya, dan menampilkan halaman web kepada pengguna.
